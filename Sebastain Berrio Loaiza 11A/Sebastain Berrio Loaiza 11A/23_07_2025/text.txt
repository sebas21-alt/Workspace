-¿Que es un framework CSS en un diseño web?
R:Un framework CSS es una colección predefinida de estilos, componentes y reglas CSS que facilitan el desarrollo web. 
En lugar de escribir código CSS desde cero, los frameworks ofrecen una base con estilos y estructuras ya listas, 
como sistemas de cuadrícula, componentes de interfaz de usuario y estilos para elementos comunes como botones y tipografía.
Esto permite a los desarrolladores crear sitios web de manera más rápida, consistente y eficiente.

-¿que es bootstrap en diseño web?

R:Bootstrap es un framework CSS de código abierto para desarrollo web que facilita la creación de diseños responsivos y atractivos.
 Proporciona una colección de componentes y herramientas pre-diseñadas, como sistemas de cuadrícula, botones, formularios y barras de navegación, 
que permiten a los desarrolladores construir sitios web y aplicaciones web de manera más rápida y eficiente.
