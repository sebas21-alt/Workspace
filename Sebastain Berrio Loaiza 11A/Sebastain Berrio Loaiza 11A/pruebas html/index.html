sistemas de medida en CSS

  • pixcels(px):Unidad más común, representa un punto en la pantalla.
  • Puntos (pt): Unidad utilizada tradicionalmente para tipografía, 1 pt = 1/72 de pulgada.
  • Pulgadas (in): Unidad de medida de longitud.
  • Centímetros (cm): Unidad del sistema métrico.
  • Milímetros (mm): Unidad del sistema métrico.
